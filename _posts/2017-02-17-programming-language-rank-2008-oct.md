TIOBE网站刚刚发布了2008年10月份编程语言排行榜。该网站统计数据主要来源是各大搜索引擎的统计数据，反应当前编程语言的流行程度及关注度，而并不是&#8220;最棒的&#8221;或&#8220;产生代码量最多的&#8221;编程语言。

本月名列前十的编程语言是：

  1. Java
  2. C
  3. C++
  4. VB
  5. PHP
  6. Python
  7. C#
  8. Delphi
  9. Perl
 10. JavaScript
