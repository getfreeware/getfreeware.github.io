First Impression 是一款小巧而快速的图片浏览器。它舍弃了普通 Windows 程序的UI界面，在使用时候可以用键盘控制，其特点包括：

  * 没有 UI ！从而争取最大的显示空间！
  * 快速而简易的键盘导航；
  * 速度感！
  * Unicode 支持；
  * 提供常用的图片处理功能：翻转，旋转，缩放等。
